# Tech History & Maintainers

## Java & Spring Boot
- **Spring Boot**: 
    - **Launch**: v1.0 released in April 2014 by Pivotal (now VMware Tanzu).
    - **History**: Designed to simplify Java development by providing a "opinionated" view of the Spring platform. Led by Phil Webb and Dave Syer.
    - **Maintainers**: VMware.

## Apache Kafka
- **Launch**: Open-sourced in 2011 (v0.7 in May 2011).
- **History**: Developed at LinkedIn by Jay Kreps, Neha Narkhede, and Jun Rao to handle real-time data feeds.
- **Maintainers**: Apache Software Foundation (Community) and Confluent (Commercial).

## PostgreSQL
- **Launch**: v6.0 (first as PostgreSQL) in January 1997.
- **History**: Roots in the Ingres project (1982) and POSTGRES at UC Berkeley (1986).
- **Maintainers**: PostgreSQL Global Development Group.

## Docker
- **Launch**: Publicly unveiled in March 2013 at PyCon.
- **History**: Started by Solomon Hykes at dotCloud. Rebranded to Docker, Inc. in 2013.
- **Maintainers**: Docker, Inc. and the Moby project.

## Resilience4j
- **History**: Developed as a lightweight fault tolerance library for Java 8, gaining popularity as an alternative to Netflix Hystrix.
- **Maintainers**: Community-driven open-source project.

## OpenFeign
- **History**: Declarative HTTP client that simplifies writing web service consumers in Java.
- **Maintainers**: Community-driven, part of the OpenFeign organization.
