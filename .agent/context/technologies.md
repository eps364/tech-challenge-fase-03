# Technologies - Tech Challenge Fase 03

## Core Stack
- **Java 21**: Main programming language.
  - [Documentation](https://docs.oracle.com/en/java/javase/)
- **Spring Boot 3.4.1**: Framework for microservices.
  - [Documentation](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/)
- **PostgreSQL 15**: Relational database for persistence.
  - [Documentation](https://www.postgresql.org/docs/)
- **Apache Kafka (latest)**: Message broker for asynchronous communication.
  - [Documentation](https://kafka.apache.org/documentation/)

## Resilience & Integration
- **Resilience4j (Spring Boot 3 optimized)**: Implementation of Circuit Breaker, Retry, and Timeouts.
  - [Documentation](https://resilience4j.readme.io/docs)
- **Spring Cloud OpenFeign (2024.0.0 BOM)**: Declarative REST client for service-to-service and external integration.
  - [Documentation](https://docs.spring.io/spring-cloud-openfeign/docs/current/reference/html/)
- **Spring Cloud Gateway (2024.0.0 BOM)**: API Gateway functionality.
  - [Documentation](https://docs.spring.io/spring-cloud-gateway/docs/current/reference/html/)
- **Spring Cloud Netflix Eureka (2024.0.0 BOM)**: Service discovery and registration.
  - [Documentation](https://docs.spring.io/spring-cloud-netflix/docs/current/reference/html/)

## Infrastructure
- **Docker & Docker Compose**: Containerization and environment orchestration.
  - [Documentation](https://docs.docker.com/)
- **Maven**: Build and dependency management (using Maven Wrapper).
  - [Documentation](https://maven.apache.org/guides/index.html)
- **RabbitMQ 3-management**: Message broker for internal orchestration.
- **Redis 7-alpine**: Caching and session management.

## Security
- **Spring Security + JWT (Spring Boot 3.4.1 optimized)**: Framework for authentication and authorization.
  - [Documentation](https://docs.spring.io/spring-security/reference/index.html)
- **Keycloak 26.5.4**: Identity and Access Management.
