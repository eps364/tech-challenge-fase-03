Problem Type: Product Not Found

Type URI/errors/product-not-found
HTTP Status404 Not Found
StandardRFC 7807 — Problem Details for HTTP APIs

Description

This error occurs when a requested product cannot be found in the catalog. The detail field in the response body will contain the product ID that was not found.

Example Response

{
  "type": "/errors/product-not-found",
  "title": "Product Not Found",
  "status": 404,
  "detail": "Product not found: 42"
}

How to Resolve

  • Check that the product ID in the request path is correct.
  • Use GET /products to list all available products and their IDs.
